Deployment Guide

Prerequisites

  • Telegram Bot Token from @BotFather
  • Your Telegram User ID (get from @userinfobot)
  • NeonDB PostgreSQL Database (Free Tier)
  • MongoDB Atlas Cluster (Free Tier)
  • Render Account (Free Tier)
  • TMDB API Key (optional, for movie/TV features)

Step 1: Database Setup

📊 PostgreSQL (NeonDB)
  1. Create account at neon.tech
  2. Create new project
  3. Copy connection string (starts with postgresql://)
  4. Format: postgresql://user:password@host/database?sslmode=require

Used for:

  • Link provider data · Force-sub channels · Generated links & clone bots
  • Broadcast history · Upload progress · Category settings
  • Group management modules · Notes, warns, bans, filters
  • Welcome/goodbye messages · All SQL-backed features
🍃 MongoDB (Atlas)
  1. Create account at mongodb.com
  2. Create free cluster (M0)
  3. Create database user
  4. Whitelist all IPs (0.0.0.0/0)
  5. Get connection string
  6. Format: mongodb+srv://user:password@cluster.mongodb.net/database

Used for:

  • Poster premium plans · Daily usage tracking
  • Couples system · Chatbot data · User analytics
  • Speed-optimized queries

  1. Fork/Clone Repository

    git add .
